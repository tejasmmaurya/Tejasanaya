## üöÄ GitHub Pages Deployment

This repository is configured to automatically deploy to GitHub Pages!

### Setup Instructions

1. **Merge this PR** to the `main` branch
2. **Enable GitHub Pages** in your repository settings:
   - Go to your repository on GitHub
   - Click on **Settings** tab
   - Scroll down to **Pages** section (in the left sidebar under "Code and automation")
   - Under **Source**, select **GitHub Actions**
3. **Wait for deployment** - The GitHub Actions workflow will automatically deploy your site
4. **Access your site** at: `https://tejasmmaurya.github.io/Tejasanaya/`

### How It Works

- `index.html` - Your main HTML page (previously Tejas.html)
- `.github/workflows/static.yml` - GitHub Actions workflow that automatically deploys to Pages

### Manual Deployment

You can also trigger a manual deployment:
1. Go to the **Actions** tab in your repository
2. Select **Deploy static content to Pages**
3. Click **Run workflow**
